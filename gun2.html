AI Agent Geliştirme Temelleri

60 dk
Kod yazmadan yazılım üretmek ne demek? AI Agent kavramı, niyet-odaklı üretim mantığı ve "isteyen ama yöneten" zihniyeti. Canlı kanıt: gezdiğin bu site de AI Agent'larla yapıldı — görselleri dahil.
02

Claude Code ile İlk Prompt

60 dk
Claude Code kurulumu ve ilk "işe alım": görev tarifi, task decomposition, dosya sistemiyle çalışma. İlk promptunla ilk mini uygulamanı üretiyorsun.
03

Kendi AI Agent'ını Kur

60 dk
Kendi işine özel bir AI Agent tasarlıyorsun: kimliği, görevi, hafızası. Sosyal medya asistanı, müşteri takipçisi, rapor yazarı — sen seç, birlikte kuralım.
04

Canlı Uygulama

60 dk
Herkes kendi Agent'ını canlı çalıştırır, ürettiklerini görür, iyileştirir. Takıldığın yerde birlikte çözüyoruz — gerçek atölye, gerçek üretim.

Yayınla ve İzle — GitHub + Dashboard

60 dk
